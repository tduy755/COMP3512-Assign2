F1 Dashboard Project

This project allows users to view Formula 1 race data for previous seasons, providing a look at race results, qualifying results, and race data. The user interface allows sorting of race data by various criteria such as position or constructor, and also features modal popups displaying additional details about drivers, constructors, and circuits. We built this project using HTML, CSS, and JavaScript to create a responsive and functional front-end design. The app was a team effort by Duy Tran and Chase Knight, and you can find the code on GitHub. View the GitHub Repository
